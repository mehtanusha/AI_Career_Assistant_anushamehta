# 🚀 AI Career Assistant

An intelligent career guidance platform built using **Python**, **Streamlit**, and **Hugging Face LLMs** that helps students and professionals plan their careers, prepare for placements, practice interviews, and optimize their resumes for ATS systems.

---

## ✨ Features

### 🏠 Dashboard

* Interactive AI career chatbot
* Ask career-related questions
* Get personalized career guidance using LLMs

### 🎯 Career Roadmap Generator

* Generate a complete roadmap for any career
* Get recommended:

  * Skills
  * Technologies
  * Learning paths
  * Projects
  * Certifications
  * Interview preparation strategies

### 📊 Placement Predictor

* Predict placement chances based on:

  * CGPA
  * Technical skills
* Receive personalized improvement suggestions

### 🎤 Interview Question Generator

* Generate interview questions with answers
* Supports any technology/domain
* Useful for placement and job preparation

### 📄 ATS Resume Analyzer

* Upload PDF or TXT resumes
* Compare resume with job descriptions
* Generate:

  * ATS Match Score
  * Matching Skills
  * Missing Skills
  * Missing Keywords
  * Resume Strengths
  * Resume Weaknesses
  * Improvement Suggestions
  * Interview Readiness Report

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/AI-Career-Assistant.git

cd AI-Career-Assistant
```

### Create Virtual Environment

```bash
python -m venv venv
```

Activate environment:

**Windows**

```bash
venv\Scripts\activate
```

**Linux/Mac**

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file in the project root directory:

```env
HF_TOKEN=your_huggingface_api_token
```

Get your Hugging Face token from:

https://huggingface.co/settings/tokens

## ▶️ Run the Application

```bash
streamlit run app.py
```
